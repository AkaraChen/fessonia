    DefaultLogger(context, _parent, _opts)

    Class representing a Console-based default logger.

    You can replace this logger with any logger that supports the following methods:

    • trace
    • debug
    • info
    • warn
    • error
    • fatal

    To specify a different logger, specify a logger option when using getConfig.
